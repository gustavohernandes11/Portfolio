## Rodando localmente

Clone o projeto

```bash
  git clone "https://github.com/gustavohernandes11/portfolio/" portfolio
```

Entre no diretório do projeto

```bash
  cd portfolio
```

Instale as dependências

```bash
  npm i
```

Inicie o servidor

```bash
  npm run dev
```
